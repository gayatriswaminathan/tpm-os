# Example: launch announcement + video brief

Shows the `launch-announcement` skill turning a launch into a written announcement and a short video script.

## Input

```
moment: the new unified intake platform is live for all teams
audience: whole engineering org (internal)
the one thing: there's now a single front door for all requests
key facts:
  - replaces 6 separate team-by-team intake forms
  - one form, auto-routed to the right team
  - average response time down from 5 days to 1
next action: submit your next request through it
tone: celebratory but practical
format: written + 20-second video, 16:9
```

## Output

### Part A — Written announcement

**One front door for every request — intake is now unified.**
No more hunting for the right form or the right team. Starting today, every request goes through a single intake — one form, automatically routed to the team that owns it. We retired six separate intake paths, and early teams are seeing responses in about a day instead of five.

👉 Submit your next request through the new intake (link) — that's the only step you need to change.

### Part B — Video script & brief

```
# Unified Intake is Live — 0:20 launch video
Audience: engineering org · Tone: upbeat, practical · Aspect: 16:9

Scene 1 (0:00–0:04) — The hook
  On-screen text: "Six forms. Five-day waits. Which team owns this again?"
  Visual: a cluttered desktop with six near-identical request forms scattered, a frustrated cursor hovering

Scene 2 (0:04–0:11) — The reveal
  On-screen text: "One front door. Every request."
  Visual: the six forms slide together and collapse into a single clean intake form, centered, calm

Scene 3 (0:11–0:17) — The proof
  On-screen text: "Auto-routed. ~1 day, not 5."
  Visual: a request enters the single form and an arrow routes it instantly to the right team; a "5 days → 1 day" counter ticks down

Scene 4 (0:17–0:20) — The ask
  On-screen text: "Submit your next request here."
  Visual: clean CTA card with the intake link and the program logo

Voiceover (optional): "Intake just got simple. One form, routed for you."
Music/feel: confident, light, modern
```

---

Notice what the skill did: it picked **one** takeaway (single front door), led with the audience's benefit (no more hunting / faster responses) rather than the team's milestone, ended with one clear action, and turned the same facts into a tight 4-scene video brief where every "Visual" line is a ready prompt for an AI video tool or editor.
